# Roadmap — guitar-chords

## Pendiente / Pending

### Cifrado español (Do-Re-Mi) / Spanish solfège notation
Toggle para cambiar la notación de acordes de inglés (C-D-E) a solfège (Do-Re-Mi-Fa-Sol-La-Si).
Útil para guitarristas hispanohablantes que aprenden de partituras locales.

Toggle to switch chord notation from English (C-D-E) to solfège (Do-Re-Mi-Fa-Sol-La-Si).
Useful for Spanish-speaking guitarists who learn from local sheet music.

---

### Búsqueda difusa ordenada por relevancia / Fuzzy search ranked by relevance
La búsqueda actual es prefijo exacto. Agregar fuzzy matching con ranking por relevancia
(e.g. distancia de edición, popularidad) para que `amaj` sugiera `Amaj7` aunque
el usuario escriba diferente.

Current search is exact-prefix. Add fuzzy matching with relevance ranking
(e.g. edit distance, popularity) so that `amaj` suggests `Amaj7` even with typos.

---

### Filtros de búsqueda mejorados / Improved search filters UI
La presentación actual de las píldoras de filtro es funcional pero básica.
Mejorar la UX: agrupar, mostrar conteos, soporte de selección múltiple, o
convertirlos en un panel lateral en pantallas grandes.

The current filter pills are functional but basic.
Improve UX: grouping, showing counts, multi-select support, or a side panel on large screens.
